# 🚀 Vaishal Coding Practice

A collection of Python programming exercises, pattern programs, number-based logic problems, and searching algorithms created for learning, practice, and interview preparation.

## 📌 About

This repository contains Python programs covering:

* Number-based coding problems
* Pattern printing programs
* Searching algorithms
* Basic logic-building exercises
* Problem-solving practice for coding interviews
* Python fundamentals and syntax practice

The goal of this repository is to strengthen programming fundamentals and improve problem-solving skills through consistent coding practice.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/VaishalMalu/Vaishal-Coding-Practice.git
```

2. Navigate to the project directory:

```bash
cd Vaishal-Coding-Practice
```

3. Run any Python file:

```bash
python filename.py
```

Example:

```bash
python pattern1.py
```
